### 世界并非如你所见。

我们的未来岌岌可危，你必须选择加入一方。

欧洲一队科学家发现了一种神秘的能量，这一能量的来源和用途无人知晓，但一些研究人员认为它会影响我们的思维方式。

我们必须控制住这股力量，否则就会被其奴役。

> <span style="color:#03DC03;">启蒙军</span> (The Enlightened)
> 试图接纳这股力量，坚信这是对人类的恩赐。

> <span style="color:#0088FF;">反抗军</span> (The Resistance)
> 奋力抵抗并保护我们人类剩余的资源和财富。

安装 Ingress 来改变世界吧。
